## Overview

**libft** is a static library (`libft.a`) written in C that provides:

- Re-implementations of common `<string.h>`, `<ctype.h>`, `<stdlib.h>` functions.
- Additional helper functions for string manipulation, memory handling, and data structures.
- A simple linked list implementation and related utilities (bonus part).

This project is the foundation for many subsequent 42 projects, as it provides a reusable set of tools that you can link against in your future work.

## Features

### Libc Functions

Re-implemented versions of standard C functions:

- **Character checks & conversion**
  - `ft_isalpha`
  - `ft_isdigit`
  - `ft_isalnum`
  - `ft_isascii`
  - `ft_isprint`
  - `ft_toupper`
  - `ft_tolower`

- **Memory**
  - `ft_memset`
  - `ft_bzero`
  - `ft_memcpy`
  - `ft_memmove`
  - `ft_memchr`
  - `ft_memcmp`
  - `ft_calloc`

- **Strings**
  - `ft_strlen`
  - `ft_strlcpy`
  - `ft_strlcat`
  - `ft_strchr`
  - `ft_strrchr`
  - `ft_strncmp`
  - `ft_strnstr`
  - `ft_strdup`

### Additional Functions

Utility functions that are not part of the original libc, but are very handy:

- **String handling**
  - `ft_substr`
  - `ft_strjoin`
  - `ft_strtrim`
  - `ft_split`
  - `ft_itoa`
  - `ft_strmapi`
  - `ft_striteri`

- **I/O**
  - `ft_putchar_fd`
  - `ft_putstr_fd`
  - `ft_putendl_fd`
  - `ft_putnbr_fd`

> The exact list may vary slightly depending on your project version, but the core idea remains the same.

### Bonus (Linked List) Functions

If the **bonus** part is completed, the library also includes a basic singly linked list implementation:

- `t_list` structure:
  ```c
  typedef struct s_list
  {
      void            *content;
      struct s_list   *next;
  }   t_list;
  ```

- **List creation & basic ops**
  - `ft_lstnew`
  - `ft_lstadd_front`
  - `ft_lstsize`
  - `ft_lstlast`
  - `ft_lstadd_back`

- **List deletion & iteration**
  - `ft_lstdelone`
  - `ft_lstclear`
  - `ft_lstiter`
  - `ft_lstmap`

## Installation

Clone the repository and build the library:

```bash
git clone <your-repo-url> libft
cd libft
make
```

This will generate the static library:

```text
libft.a
```

To clean object files:

```bash
make clean
```

To clean object files and the library:

```bash
make fclean
```

To recompile from scratch:

```bash
make re
```

If you implemented the bonus part, you can build it with:

```bash
make bonus
```

## Usage

To use **libft** in another C project:

1. **Include the header** in your source files:
   ```c
   #include "libft.h"
   ```

2. **Compile your program** and link with `libft.a`:

   ```bash
   gcc -Wall -Wextra -Werror main.c -L. -lft -I. -o my_program
   ```

   Or if the library is in a different path:

   ```bash
   gcc -Wall -Wextra -Werror main.c -L./libft -lft -I./libft -o my_program
   ```

Adjust `-I` and `-L` according to where `libft.h` and `libft.a` are.

## Examples

### Example 1: Using `ft_split`

```c
#include "libft.h"
#include <stdio.h>

int main(void)
{
    char    **words;
    int     i;

    words = ft_split("Hello world from libft", ' ');
    if (!words)
        return (1);
    i = 0;
    while (words[i])
    {
        printf("%s\n", words[i]);
        free(words[i]);
        i++;
    }
    free(words);
    return (0);
}
```

### Example 2: Using the linked list API (bonus)

```c
#include "libft.h"
#include <stdio.h>

void print_content(void *content)
{
    printf("%s\n", (char *)content);
}

int main(void)
{
    t_list *list = NULL;

    ft_lstadd_back(&list, ft_lstnew("first"));
    ft_lstadd_back(&list, ft_lstnew("second"));
    ft_lstadd_back(&list, ft_lstnew("third"));

    ft_lstiter(list, &print_content);
    ft_lstclear(&list, NULL);
    return (0);
}
```
